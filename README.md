# Employee Payroll System (Java + JDBC)

A console-based application to manage employee salary details.

## Features
- Add Employee
- View Employees
- Generate Payslip
- Delete Employee

## Technologies
- Java
- JDBC
- MySQL

## Concepts
- OOP
- CRUD Operations
- Layered Architecture

## How to Run
1. Create database in MySQL
2. Update DB credentials
3. Compile and run Main.java
